**Matplotlib** is a popular Python library used for **data visualization**.
It helps convert numerical data into graphical form so that information can be understood easily.

Instead of printing values in the terminal, we can represent them using graphs and charts.

### Features
- Create line graphs
- Bar charts
- Pie charts
- Histograms
- Customize labels, titles and colors

### Why it is useful
Data visualization helps to:
- Analyze patterns
- Compare values
- Understand trends quickly
- Present results clearly in projects

### Example Use Cases
- Student marks analysis
- Sensor data monitoring
- IoT data visualization
- Statistical analysis
